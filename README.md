## Get started
* Create new folder for your project
* Clone repository 
```sh
git clone https://github.com/drupalspec/starter.git
```
* MV all files from starter folder in root folder
```sh
cd starter
```
```sh
mv * ..
```
* Remove starter folder
```sh
rm -r -f starter
```

## Project tree
* src - source files
* build - compile files

## How to use it
* Call npm i command
* Call gulp command
* Open browser on localhost:40000
